# 🏦 LedgerVault

A modern full-stack ledger-based banking platform built with React, Node.js, Express, and MongoDB.

LedgerVault simulates real-world banking workflows with secure authentication, multi-account management, money transfers, transaction statements, and real-time balance tracking.

---

# ✨ Features

- JWT Authentication & Protected Routes
- Multi-Account Management
- Ledger-Based Transactions
- Send Money Workflow
- Real-Time Balance Calculation
- Transaction History & Statements
- System Deposit Handling
- Dark / Light Theme
- Responsive Dashboard UI
- REST API Architecture

---
